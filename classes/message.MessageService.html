    init

    • init(channelName: string): Promise<{ channelId: number }>

      Initializes the messaging system by joining a channel and setting up subscriptions.

      • Joins the specified channel and stores its ID.
      • Fetches and stores member details in memberDetailsMap.
      • Retrieves and formats channel messages, storing them by topic.
      • Subscribes to member updates for the channel.
      • Subscribes to message updates, starting from the last retrieved message.

    subscribeToTopic

    • subscribeToTopic<Topic extends Topics>(
          topic: Topic,
          callback: MessageCallback<Topic>,
          isStreamed?: boolean,
      ): void

      Subscribes a callback function to receive messages for a specific topic.

      • If the channel is not available, it immediately invokes the callback with an error.
      • If there is an existing subscription error, it informs the caller to reinitialize.
      • Triggers the callback with existing messages for the topic.
      • Stores the callback and its streaming preference in topicCallbacksMap.
      • Multiple callback functions per topic are supported.
